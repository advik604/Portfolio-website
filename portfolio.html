Tindog

is a beautiful website that serves as a static site for a dating web app similar to Tinder, but for dogs. It was developed using HTML, CSS, and Bootstrap . The main framework used in building the website is Bootstrap, along with its components like Carousel, Card, Buttons, and Navbar. It also incorporates HTML5, CSS3, Google Fonts, and Font Awesome .

Drum-Kit

is an interactive and engaging virtual drum kit. It was developed using HTML, CSS, and JavaScript. The main purpose of this website is to provide users with a fun and immersive drumming experience. Upon visiting the website, users are presented with a virtual drum kit interface that closely resembles a physical drum set. Each drum element, such as the snare, bass, hi-hat, and cymbals, is represented by a clickable button on the screen.

Student management system

The Student Record Management System is a C++ project that allows users to effectively manage student data. It serves as a management information system for education, providing features like creating, reading, modifying, and deleting student records.Using concepts such as structures, loops, arrays, conditionals (if-else and switch), and more, this project enables users to store student data and perform actions like displaying, updating, deleting, and searching for records.
